
# JAXE Creator App

<!-- This is a React Native + Expo 54 mobile application for TikTok LIVE creators -->

A modern mobile application built with React Native and Expo for managing creator activities, academy courses, battles, and earnings.

## Features

<!-- Core features include: Academy courses with video progress tracking, Battle scheduling, Challenge system, Bonus tracking, Manager portal, and real-time notifications -->

- **Academy**: Video courses with progress tracking and quizzes
- **Battles**: Schedule and manage creator battles
- **Challenges**: 21-day challenge system with daily objectives
- **Bonuses**: Track earnings and graduation milestones
- **Manager Portal**: Manager dashboard for overseeing creators
- **Profile**: Manage creator profile and settings
